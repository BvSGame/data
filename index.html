Here you can find publicly accessible raw data that may be static or dynamic. If you don't understand what this place is for, go to home page. Please.

Otherwise, if you know what you're doing and why, here's what you can grab:

Directories

  • milestone/ ✧ Contains version history and change logs.
  • wiki/ ✧ Contains wiki files in Markdown format.

Files

  • LICENSE ✧ Open-source license information.
  • Version files:
    • version ✧ Simple text file indicating full version number.
    • version_float ✧ Simple text file indicating version number as a float number.
    • version_short ✧ Simple text file indicating shortened version number.
    • version.json ✧ JSON file with detailed info about current version.

You can also go to the official repositories on Codeberg, sourcehut and GitHub, if you want to.
