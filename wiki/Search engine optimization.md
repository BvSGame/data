## Nuances of SEO

Some search engines may implement **IndexNow** feature. [See their documentation](https://www.indexnow.org/documentation). Particularly, IndexNow supported by [Bing](https://www.bing.com/indexnow) and [Yandex](https://yandex.com/support/webmaster/en/indexing-options/index-now).

### Specific to search engines

- **Bing**:
	- There are [Bing Webmaster Tools](https://www.bing.com/webmasters/about). #NotTested.
- **Brave Search**:
	- For indexing in Brave Search, Brave Browser has “Web Discovery Project” setting. With this setting enabled, basically you have to browse the web using Brave Browser as you would normally do, and Brave Browser will do anonymous pings to Brave Search, therefore potentially triggering indexing of webpages.
		- You can find “Web Discovery Project” setting in Brave Browser using this URL:
		  ```txt
			brave://settings/?search=Web+Discovery+Project
		  ```
- **DuckDuckGo**:
	- DuckDuckGo mainly relies on Bing’s crawler. Therefore, see Bing.
- **Google Search**:
	- There is [Google Search Console](https://search.google.com/search-console) that is able to read [sitemaps](Sitemap.md).
