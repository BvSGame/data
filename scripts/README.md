## Notes

### Units of measurement

A measure has been taken to adapt this script to different countries.  Where you can see **the units of measurement**, you may see clarifications in **square brackets** where the values are converted for better understanding.  For example: **“100 meters [109.3 yd]”**.  In dialogs, **don’t read** the contents of the square brackets aloud.
